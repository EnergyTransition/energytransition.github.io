PipeDiameterEnum

Defines the pipe diameter in DN size. When this field is set to VALUE_SPECIFIED (default) both innerDiameter and outerDiameter fields are used for diameter specification. If not, an appropriate pipe catalog should be used to calculate the properties of the pipe, based on the speficied DN size. This allows for different pipe types, e.g. with or without insulation.

VALUE_SPECIFIED - Default option for this Pipe diameter: it specifies that the innerDiameter and outerDiamter fields are used instead of values from this enumeration.
