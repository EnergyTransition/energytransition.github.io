AssetStateEnum

The AssetStateEnum allows to specify the ‘state’ of an Asset, mainly used for manually simulating different scenarios. It can be used to temporarily disable an Asset for a specific simulation, without the need to remove the Asset and loose all its parameter configurations. By default the value is ‘ENABLED’. ‘DISABLED’ can be used to disable an Asset temporarily, ‘OPTINAL’ can be used to specify to a simulator that the Asset can optionally be placed here (to be decided by the simulator, algorithm or tool).
