Abstract class to define the shape/location of an asset or area. Parent class of e.g. Point, Line and Polygon

Name Description
Point Describes a point geometry, which can be used for giving assets a location on a map

Polygon Describes a polygon geometry, which can be used for defining the shape of an area or building
Line Describes a line geometry based on a list of points, which can be used to define the shape of pipes and cables
